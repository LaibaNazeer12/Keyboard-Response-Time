# ⌨️ Keyboard Response Test (HCI Lab)

## 📌 Overview

This project is a simple Human-Computer Interaction (HCI) lab experiment that measures a user's **reaction time**.

A random **letter or number** is displayed on the screen, and the user must press the correct key as quickly as possible.

---

## 🎯 Objective

* Measure user reaction time
* Understand human response behavior
* Practice basic HCI concepts

---

## ⚙️ How It Works

1. Click the **Start** button
2. A random **letter or number** appears
3. Press:

   * **A** → if it is a letter
   * **L** → if it is a number
4. The system calculates:

   * Reaction time (in milliseconds)
   * Average reaction time
